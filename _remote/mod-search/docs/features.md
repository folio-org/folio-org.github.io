# Module Features

This module provides the following features:

| Feature                                                            | Description                                                                                               |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| [Authority Browse](features/authority-browse.md)                 | Bidirectional alphabetical browse of authority headings.                                                  |
| [Authority Search](features/authority-search.md)                   | CQL-based search over authority records with optional linked-title counts.                                |
| [Call Number Browse](features/call-number-browse.md)              | Bidirectional alphabetical browse of instance records by call number type.                                |
| [Classification Number Browse](features/classification-browse.md) | Bidirectional alphabetical browse of instance records by classification scheme.                           |
| [Configuration](features/configuration.md)                         | Runtime management of search features, indexing languages, and browse option configurations.              |
| [Consortium Search](features/consortium-search.md)                 | Cross-tenant search for consolidated holdings, items, and locations in an ECS consortium.                 |
| [Contributor Browse](features/contributor-browse.md)              | Bidirectional alphabetical browse of contributor names linked to instances.                               |
| [CQL Query Translation](features/cql-query.md)                     | Translation of CQL queries into Elasticsearch DSL for all search and browse endpoints.                    |
| [Field Mapping Engine](features/field-mapping-engine.md)           | Core infrastructure for resource field definitions, Elasticsearch mappings, and field processor pipeline. |
| [Indexing and Messaging](features/indexing-and-messaging.md)       | Real-time index updates via Kafka event consumers and manual record indexing REST API.                    |
| [Instance Search](features/instance-search.md)                     | CQL-based search over bibliographic instance records with facets and async ID streaming.                  |
| [Linked Data Search](features/linked-data-search.md)               | CQL-based search over Linked Data works, instances, and hubs.                                             |
| [Reindex](features/reindex.md)                                     | Full and partial reindex orchestration for inventory instance records with status tracking.               |
| [Subject Browse](features/subject-browse.md)                      | Bidirectional alphabetical browse of subject headings linked to instances.                                |
